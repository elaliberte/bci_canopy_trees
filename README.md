# BCI Canopy Tree species list

This repo compiles different data sources to build a checklist of all known canopy tree species on Barro Colorado Island, Panama.

The goal is then to compare to our own data on labelled close-up drone photos of canopy trees in this whole-island BCI mapping effort (<https://arcg.is/0uWLK4>) to determine the extent to which we've sampled most or all canopy tree species, or not.
